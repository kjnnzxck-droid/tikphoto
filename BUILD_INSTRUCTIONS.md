# 🚀 تعليمات بناء TikPhoto APK\n\n## الخطوة 1: تثبيت المتطلبات الأساسية\n\n### على Windows:\n```bash\n# تثبيت Python (3.9 أو أحدث)\n# من https://www.python.org/downloads/\n\n# تثبيت المكتبات المطلوبة\npip install buildozer\npip install cython\npip install -r requirements.txt\n```\n\n### على macOS:\n```bash\nbrew install python3\npip3 install buildozer\npip3 install cython\npip3 install -r requirements.txt\n```\n\n### على Linux (Ubuntu/Debian):\n```bash\nsudo apt-get update\nsudo apt-get install python3 python3-pip python3-dev git\nsudo pip3 install buildozer cython\npip3 install -r requirements.txt\n```\n\n## الخطوة 2: تثبيت Java Development Kit (JDK)\n\n### Windows:\n- حمّل JDK من: https://www.oracle.com/java/technologies/downloads/\n- أو استخدم: `choco install jdk`\n\n### macOS:\n```bash\nbrew install openjdk@11\n```\n\n### Linux:\n```bash\nsudo apt-get install openjdk-11-jdk\n```\n\n## الخطوة 3: تثبيت Android SDK\n\n```bash\n# تحميل من https://developer.android.com/studio\n# أو استخدام sdkmanager\n\n# بعد التثبيت، حدد مسار SDK:\nexport ANDROID_SDK_ROOT=/path/to/android-sdk\nexport ANDROID_HOME=/path/to/android-sdk\n```\n\n## الخطوة 4: بناء ملف APK\n\n### الطريقة الأساسية:\n```bash\n# في مجلد المشروع\ncd /path/to/tikphoto\nbuildozer android debug\n```\n\n### مع ملف buildozer مخصص:\n```bash\nbuildozer -f buildozer.spec android debug\n```\n\n### للبناء الإنتاجي (Release):\n```bash\nbuildozer android release\n```\n\n## الخطوة 5: الملف النهائي\n\nسيتم إنشاء ملف APK في:\n```\nbin/tikphoto-1.0.0-debug.apk\n```\n\n## الخطوة 6: تثبيت على الهاتف\n\n### باستخدام ADB:\n```bash\n# تأكد من توصيل الهاتف بالكمبيوتر\nadb devices\n\n# تثبيت التطبيق\nadb install bin/tikphoto-1.0.0-debug.apk\n```\n\n### نقل مباشر:\n1. انسخ الملف `bin/tikphoto-1.0.0-debug.apk`\n2. انقله إلى الهاتف\n3. انقر عليه لتثبيته\n\n## المتطلبات على الهاتف\n\n- نظام Android 5.0 (API 21) أو أحدث\n- 50 MB مساحة تخزين حرة\n- الإنترنت (للتثبيت الأول)\n\n## الأذونات المطلوبة\n\n- 📷 الكاميرا (التقاط صور)\n- 💾 الوصول للملفات (قراءة وحفظ الصور)\n- 📍 الموقع الجغرافي (اختياري)\n\n## حل المشاكل الشائعة\n\n### مشكلة: \"buildozer: command not found\"\n```bash\npip install --upgrade buildozer\n```\n\n### مشكلة: \"Android SDK not found\"\n```bash\n# حدد مسار SDK\nexport ANDROID_SDK_ROOT=/path/to/android-sdk\n```\n\n### مشكلة: \"Java not found\"\n```bash\n# تحقق من تثبيت Java\njava -version\n\n# إذا لم يكن مثبت، ثبّته\n```\n\n### مشكلة: \"Build failed\"\n```bash\n# نظّف المشروع\nbuildozer android clean\n\n# أعد المحاولة\nbuildozer android debug\n```\n\n## معلومات التطبيق\n\n- **الاسم**: TikPhoto\n- **الإصدار**: 1.0.0\n- **الحزمة**: org.tikphoto\n- **الحد الأدنى من API**: 21\n- **API المستهدف**: 33\n\n## معلومات الحقوق\n\n© 2024
