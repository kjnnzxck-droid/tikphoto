# 📸 TikPhoto - Professional Image Editor

**TikPhoto** هو تطبيق متقدم لتعديل الصور بميزات احترافية وواجهة مستخدم حديثة.

## ✨ المميزات الرئيسية

### 🖼️ أدوات التحرير:
- ✂️ قص الصور (Crop)
- 🔄 تدوير الصور (Rotate)
- 📄 تغيير حجم الصور (Resize)
- 🌡️ تعديل السطوع والتباين (Brightness & Contrast)
- 🎨 تصحيح الألوان (Color Correction)
- 🖌️ أدوات الرسم والكتابة (Drawing & Text)
- ✨ الفلاتر الاحترافية (Filters)
- 🌈 تأثيرات بصرية متقدمة (Effects)
- 🖼️ إضافة إطارات ونصوص (Frames & Text)
- 💧 تأثيرات خاصة (Special Effects)

### 📱 ميزات التطبيق:
- واجهة مستخدم حديثة وسهلة الاستخدام
- دعم جميع تنسيقات الصور الشهيرة
- معاينة فورية للتأثيرات
- حفظ الصور بجودة عالية
- مشاركة سريعة على وسائل التواصل

## 🛠️ التكنولوجيا المستخدمة

- **Python 3.9+**
- **Kivy Framework** - للواجهات الرسومية
- **OpenCV** - لمعالجة الصور
- **Pillow** - لتحرير الصور
- **NumPy** - للعمليات الرياضية

## 📦 متطلبات التثبيت

```bash
pip install kivy opencv-python pillow numpy buildozer
```

## 🚀 البدء السريع

### تشغيل على Windows/Mac/Linux:
```bash
python main.py
```

### بناء تطبيق Android:
```bash
buildozer android debug
```

2. **Sharpen** - شحذ الصورة
3. **Edge Detection** - كشف الحواف

6. **Grayscale** - تحويل إلى رمادي
